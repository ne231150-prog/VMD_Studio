# VMD Studio v1.0 要件定義書

## 1. サービス概要

VMD Studioは、アパレル店舗のVMDをバーチャル空間上で作成・確認・保存できるWebアプリである。

店舗スタッフや本部VMD担当者が、実店舗で商品を動かす前に、ラック・テーブル・マネキン・商品配置を3D上で検討できるようにする。

## 2. 目的

- VMD作業の時間短縮
- 店舗間でのVMD共有
- 本部と店舗の認識ズレ削減
- 新人スタッフへのVMD教育支援
- 将来的なAI提案機能の土台作り

## 3. v1.0で実装する機能

### 認証
- 新規登録
- ログイン
- ログアウト

### 店舗管理
- 店舗登録
- 店舗一覧表示
- 店舗編集
- 店舗画像アップロード
- 店舗レイアウト作成

### 商品管理
- 商品登録
- 商品画像アップロード
- 商品カテゴリー管理
  - トップス
  - パンツ
  - アイテム
- 商品一覧表示
- 商品検索
- 商品編集

### VMD編集
- 3D店舗表示
- ラック配置
- テーブル配置
- マネキン配置
- 商品配置
- ハンガー表示
- 畳み表示
- マネキン着用表示
- 重ね着表現
- 配置保存

### 共有・保存
- VMD案の保存
- VMD案の複製
- Before / After比較
- 将来的にPDF出力

## 4. v1.0では実装しない機能

- AIによる商品3D自動生成
- AIによる店舗3D自動生成
- AIによるVMD提案
- 売上データ連携
- AR / VR対応

これらはv2.0以降で実装する。

## 5. 想定ユーザー

- アパレル店舗スタッフ
- 店長
- VMD担当者
- 本部販売促進担当者
- ブランド運営担当者

## 6. 最重要価値

VMD Studioの価値は、AIそのものではなく、店舗VMDを事前にバーチャルで検討できることにある。

v1.0では、AIなしでも企業が使いたくなる3D VMD編集ツールを目指す。git status
